## Project Setup

Install dependencies and dev dependencies, run:

using npm

```
npm install
```

using yarn

```
yarn install
```

## Running the app

To get the app up and running, run:

using npm

```shell
npm run dev
```

using yarn

```shell
yarn dev
```

## Running Test suite

To build the app, run:

using npm

```
npm run test
```

using yarn

```
yarn test
```

## Building the app

To build the app, run:

using npm

```
npm run build
```

using yarn

```
yarn build
```
